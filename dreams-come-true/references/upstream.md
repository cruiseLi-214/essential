# Upstream reference

The product direction was informed by [Clawnify Open Studio](https://github.com/clawnify/open-studio), reviewed at commit `992e8b866d8ee40f9f295fc5d6b3c1609beacdbc`.

Open Studio is MIT licensed and demonstrates a React Flow image workflow, generation history, and mask-based editing. Draw uses an independent local-first implementation with a filesystem bridge for Codex, a general drawing scene, command synchronization, and direct `gpt-image-2` edits.
